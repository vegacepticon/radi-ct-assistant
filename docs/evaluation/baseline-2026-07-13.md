## Known issues (confirmed)

1. **RAG работает только для `conclusion`** — corpus содержит только `task: conclusion` (через `задача` field, хотя `task` в YAML не указан).
2. **Task-specific prompts не подключены** — `prompt_builder.py` всегда загружает `system_fast.txt` или `system_analytical.txt`.
3. **Reference schema не task-aware** — `_render_reference()` всегда пишет `Описание: → Заключение:`.
4. **Malformed references** — 3 файла имеют дублирующие `Заключение:` маркеры (004, 006, 008); 2 файла имеют дублирующие `Описание:` (001, 003 от 13 числа).
5. **Synthetic references** — 2 файла (2026-07-12-001, 2026-07-12-003) содержат тестовые формулировки.
6. **Area metadata некорректна** — 4 references с пустой area; `ReferenceEntry.area` возвращает только первый элемент.
7. **Все references одинакового приоритета** — нет gold/high.
8. **Нет benchmark** для измерения качества.
